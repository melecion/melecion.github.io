JAVASCRIPT

JAVASCRIPT is a programming language that is one of the core of technologies of the World Wide Web along with HTML and CSS. It has dynamic typing, prototype-based orientation and first-class functions.
Source: w3schools.com

CSS

CSS also known as Cascading Style Sheets used for describing the presentation of a document written in a mark-up language. It is designed to enable separation of presentation content (layout, colors and fonts).
Source: developer.mozilla.org

HTML

HTML or Hyper Text Markup Language, is a standard markup language for documents designed to be displayed in a web browser. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.
Source: html.com

Responsive Web Design

Responsive Web Design is an approach to web page creation that makes use of flexible layouts, flexible images and cascading style sheet media queries. The goal of responsive web design is to build web pages that detect the visitors screen size and orientation and change the layout accordingly.
Source: WhatIs.com
